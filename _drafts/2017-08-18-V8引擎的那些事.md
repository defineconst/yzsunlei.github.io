* V8 使用“常量折叠”优化技巧，导致幂（**）运算有时候不等于 Math.pow()
```javascript
Math.pow(99,99) - 99**99 的结果也不是 0 而是 -5.311379928167671e+182
```
* 严格来讲，ES2015提供了四种相等
```javascript
1、非严格相等(==)
2、严格相等(===)
3、零值相等
4、同值相等 Object.is()
```
* Object.is() 认为正零(+0)和负零(-0)不相同，但是 NaN 和 NaN 相同
